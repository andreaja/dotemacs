12 Customization

Customization for PHP mode can be done using Emacs's Customize feature1, or by adding bits of Emacs Lisp to your Emacs initialization file (~/.emacs).

M-x customize-group RET php RET
Start a `*Customize*' buffer with the customizable settings for PHP mode.
